梦里幽境 宣传帮助

2022-02-22

服务器需要适当的宣传才能吸纳更多新玩家，要保持浏览量必须经常性的对宣传帖子进行顶帖，俗话说酒香也怕巷子深，适当宣传有助于保持服务器的活跃，因为管理员属于佛系宣传，故而希望大家也能加入进来，帮助我们进行宣传顶帖。


简要操作说明

①.首先得有一个 MCBBS 论坛账号 (没有的 ☞ 点击注册 )

②.在游戏内绑定MCBBS论坛账号 (输入命令查看帮助 ☞ /bt 或 /bt help 查看更多... )

③.每次顶帖消耗80金粒 (回帖或者做任务赚取 ☞ MCBBS每日任务  ☞ MCBBS矿工茶馆)

④.论坛金粒对一般吃瓜群众基本没什么用 (金粒一般用来购买特殊道具,比如顶帖提升卡)

⑤.打开本服 MCBBS 宣传帖 (点击打开 ☞ 梦里幽境宣传帖 )

⑥.下拉到文章右下角购买并使用道具: "服务器/交易代理提升卡" 进行顶帖操作

⑦.绑定了账号并完成顶帖操作后可在线自动领取相应奖励 (奖励内容请查看服内介绍)


⑧.顶帖最佳时间为周末或者假期的早上9点左右，中午两点左右及下午五点半至晚上七点半

⑨.请不要连续顶帖，第二次顶帖必须间隔三十分钟及以上才能正常获得奖励

⑩.请注意顶帖并不是评分，四级以上账号每天可以顶帖3次，不要违规刷分顶帖，否则可能会被MCBBS封号


**********************************

具体图文步骤

「绑定操作」

➋.绑定账号及获取奖励. (务必先绑定账号,默认顶帖成功后自动领取奖励)

GUI快捷绑定:

①.根据服务器主菜单左下角图标提示或手动输入/bt打开宣传菜单,点击头颅按钮开始绑定操作

②.在聊天框中输入两次您的论坛ID [直接输入即可 不需要加/ (即名字，例如: 水煮清娃)]

③.如果没有绑定需求却不小心点击了绑定图标可以输入关键字'放弃'来取消操作

*绑错了账号或者想换一个账号需等待15天才能重新操作,也可以联系管理员协助重新绑定


命令绑定:

①.输入/bt help 查看绑定命令

*请不要绑定不属于自己的账号,以免引起不必要的纠纷,如果发现恶意绑定他人账号并引起了纠纷将双倍扣除获得的奖励


--------------------------------------------

「顶帖操作」

①.打开梦里幽境MCBBS宣传帖. (点击打开 ☞ 梦里幽境宣传帖 )

②.查阅宣传帖并下拉至右下角找到 '使用道具' 并购买顶帖道具 '服务器/交易代理提升卡'

③.使用 '服务器/交易代理提升卡' 进行一次顶帖宣传操作.

④.顶帖完成,获得奖励. (大概会出现30秒左右的检测延迟)
