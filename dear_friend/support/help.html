『聊天』
为防止聊天刷屏干扰交流，梦里幽境有聊天频道设定，一般打字 [闲聊] 聊天仅能附近200米范围内玩家可见，并不能全服可见，也就意味着你说话并非所有人都能看到，想要全服看到你的信息必须使用 [喊话] ==============================================
[喊话]命令:（小写感叹号 接上你想说的话）:
! 想说的话 （喊话需支付游戏币，一次一百梦境币，约等于4个铁矿的收购价格）

私聊某人:
/w 对方ID 内容

『金钱&交易』
付钱：
/pay ID 钱数 向某人支付现金(需二次验证)

『创建商店』:
拿着出售物品下蹲然后点击一个箱子,根据提示输入价格即可
设置收购:点击一个出售商店，然后输入/qs buy
设置出售：点击一个收购商店，然后输入/qs sell
更多商店命令请输入/qs 查看.....

『传送』
/tpa ID 请求传送到某人(ID)身边
/tpahere ID 请求某人(ID)传送到你身边

『领地』
最常用:
/res tp 地名 传送到指定领地
/res pset ID trusted t 给某人信任权限
查看圈地权限 (大小金钱等信息)
/res limits
/res 直接查看所有权限内可用命令(服务器菜单内有介绍，草方块图标内)

==============================================
自定义欢迎词(支持颜色代码):
/res message enter 内容... 来到某领地的欢迎词内容
/res message leave 内容... 离开某领地显示的内容
-----不显示欢迎词把内容留空即可
可以插入欢迎词内变量:
%player - 在提示信息中显示进出领地的玩家名
%owner - 在提示信息中显示领主
%residence - 在提示信息中显示该领地名称
